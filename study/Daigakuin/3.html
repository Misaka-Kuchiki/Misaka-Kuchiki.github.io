OpenCVに基づくビデオ顔検出

Video face detection based on OpenCV

Posted on 2019-04

1.実験要件

この実験は指定されたビデオを提供し、学生IDと等しい画像フレームの前景セグメンテーションおよび顔検出結果の出力を必要とし、各フレームで検出された顔のマトリックスフレームの位置座標を記録します。

2.実験手順

まず、指定されたビデオをロードし、フレームごとにビデオを読み取ります。そして、OpenCv3.0のBackgroundSubtractorMOG2関数メソッドを使用して、前景のセグメンテーションを実行します。

BackgroundSubtractorMOG2、このアルゴリズムも混合正規分布を基にした前景・背景の領域分割アルゴリズムです。このアルゴリズムの重要な点は，画素毎に最適な混合数を選択する点です(前の例ではアルゴリズムを通してK個の正規分布を使っていた点を思い出してください)．照明の変化などといった動的なシーンに対する適応力が優れています。

Ptr<backgroundsubtractormog2> cv::createBackgroundSubtractorMOG2 (
    int history = 500,
    double varThreshold = 16,
    bool detectShadows = true
)

historyパラメータは、背景のトレーニングに使用されるフレーム数で、デフォルトは500フレームです。この実験では、ビデオの最初の20フレームを使用します。

varThresholdパラメータは分散しきい値であり、現在のピクセルが前景か背景かを決定するために使用されます。デフォルトは16です。値が大きいほど感度が低くなります。この実験は、ビデオの特定の状況に応じて25に設定されています。

detecShadowsパラメーターは、影を検出するかどうか、デフォルトはtrue検出です。影を検出すると、プログラムの実行時間が長くなります。この実験ビデオには影が含まれていないため、falseに設定されています。
